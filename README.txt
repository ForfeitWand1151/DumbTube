
DumbTube Pro

1. Coloque sua API KEY do YouTube no arquivo app.js
2. Abra index.html no navegador

Recursos:
- Interface estilo Netflix (scroll horizontal)
- Cache local (economiza quota da API)
- Filtragem automática de Shorts
- Player otimizado para mobile
- Tentativa de manter áudio em background

Observação:
iOS limita reprodução em segundo plano em alguns casos por política do sistema.
